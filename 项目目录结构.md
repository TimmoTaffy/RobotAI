# 项目目录结构

```
RobotAI/
├── docs/ (项目文档)
│   ├── 1.系统架构设计(至主三).txt
│   ├── 2.实现路线.txt
│   ├── 3.1 数据流：从主控到世界模型.txt
│   ├── 3.2 主控输出JSON协议.txt
│   ├── 3.3 导航与控制：实现路线与细节.txt
│   ├── 4.1 测试使用指南速查.md
│   └── 4.2 连调：导航与控制.txt
│
├── src/ (核心模块)
│   ├── common/                 # 通用工具
│   ├── sensors/                # 传感器层
│   ├── transforms/             # 坐标变换
│   ├── localization/           # 定位模块
│   ├── mapping/                # 地图构建
│   ├── tracking/               # 目标跟踪
│   ├── planning/               # 路径规划
│   ├── control/                # 控制模块
│   └── viz/                    # 可视化
│
├── tests/ (测试代码)
│   ├── conftest.py                 # pytest配置
│   ├── test_path_planner.py        # 路径规划测试
│   ├── test_motion_planner.py      # 运动规划测试
│   ├── test_trajectory_tracker.py  # 轨迹跟踪测试
│   ├── test_mpc_controller.py      # MPC控制测试
│   └── test_integration.py         # 集成测试
│
├── config.json             # 主配置
├── config-guide.md         # 配置指南
│
├── main.py                 # 主程序入口
├── world_model.py          # 世界模型
│
├── run_tests.py            # 统一测试工具
├── pytest.ini              # 测试配置
├── notebooks/              #Jupyter演示
├── reports/                #测试报告
├── htmlcov/                #覆盖率报告
│
├── README.md               # 项目说明
├── requirements.txt        # 依赖包
├── 项目目录结构.md           # 全项目目录结构
└── .gitignore              # Git忽略规则
```
